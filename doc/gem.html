Heroku API Docs

Command Line | Ruby Gem | REST

Ruby Gem

Setup

require 'heroku'
heroku = Heroku.new(email, password)

Methods

list

Returns an array of app names owned by the account.

create(appname=nil)

Create an app with the provided name, or untitled if nil. Returns name of new app.

destroy(appname)

Permenantly destroy the named app.

upload_authkey(key)

Upload a new ssh public key. The key is a string and should read something like "ssh-rsa [long base64 data] user@host".

Example

require 'heroku'
heroku = Heroku.new('me@example.com', 'mypass')
heroku.create('myapp')
